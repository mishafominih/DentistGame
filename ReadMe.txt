Логика скриптов:
Миниигра
States: есть пять спрайтов, расположенные в строгом порядке(грустный -> веселый), и счетчик, изначально равный 2.
если ешь полезный фрукт, счетчик увеличивается на 1. с вредным все наоборот. Также начисляет монетки за правильную пищу
и отбирает за неправильную
Gravitation: отвечает за падение еды и обрабатывает коллизии.


Остальная часть приложения
Money: работает с монетами. В частности, работает с файлом, содержащим инфу о монетах и позволяет менять количество монет у игроков.
Вся работа с монетами только через этот скрипт!!!
Храниение данных осуществляется через PlayerPrefs. Он позваляет нам записывать данные определенного типа записывать в реестр устройства
по данному ключу.
OpenInstrument:Смотрит, какие инструменты уже открыты, а какие надо купить.
Данные хранит так же в реестре. 



Gameobject'ы:
для трех инструментов аналогично:
название инструмента: обычный игровой объект.
btn... : объект канваса, просто отображает иконку
view...: пустой объект, содержащий скрипт, связывающий два вышеописанных объекта.

Dirt1, 2, 3, 4: обычный игровой объект грязи. Просто переместить на игровое поле.
